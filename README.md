## Hoe te spelen

1. Klik op **START** om het spel te beginnen
2. Een rood lampje verschijnt op een willekeurige positie in het 4x4 grid
3. Klik zo snel mogelijk op het rode lampje
4. Na elke klik verschijnt er een nieuw lampje na een willekeurige vertraging
5. Herhaal dit 10 keer
6. Je totale tijd wordt getoond en opgeslagen in de ranking

## Features

- **4x4 Grid**: Overzichtelijk speelveld met lampjes
- **Random timing**: Variabele wachttijd tussen lampjes houdt het spannend
- **Ranking systeem**: Top 5 snelste tijden worden opgeslagen (localStorage)
- **Responsive design**: Werkt op desktop en mobile
- **Clean UI**: Moderne, kleurrijke interface

## Technologie

- [p5.js](https://p5js.org/) - JavaScript library voor creative coding
- HTML5 Canvas
- LocalStorage voor rankings

## Lokaal uitvoeren

1. Clone de repository
2. Open `index.html` in een moderne webbrowser
3. Train je reactiesnelheid!

## Tips

- Blijf gefocust op het hele grid
- Anticipeer niet te veel op de volgende positie
- Ontspan je hand tussen de klikken
